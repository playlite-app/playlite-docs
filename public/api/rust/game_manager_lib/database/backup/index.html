Module backup

Módulo de ‘backup’ e restauração de dados.

Fornece funcionalidades para exportar e importar a base de dados completa em formato JSON, incluindo biblioteca de jogos e lista de desejos.

Nota: Todas as operações usam transações ACID para garantir consistência dos dados.
