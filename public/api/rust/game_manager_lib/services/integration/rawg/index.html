Módulo de integração com a API RAWG.

Fornece funcionalidades para buscar informações sobre jogos, incluindo tendências, detalhes específicos e lançamentos futuros.

A RAWG é um banco de dados abrangente de jogos que fornece metadados, ratings, gêneros e outras informações relevantes.
