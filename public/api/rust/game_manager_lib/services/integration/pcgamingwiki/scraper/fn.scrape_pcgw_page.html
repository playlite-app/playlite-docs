Faz o scraping completo de uma página do PCGamingWiki pelo page_id.

Orquestra:

  1. Busca do wikitext via fetch_wikitext
  2. Parse dos blocos {{System requirements}}
  3. Parse dos blocos {{Game data/config}} e {{Game data/saves}}

Retorna PcgwScrapedData com todos os dados extraídos. Em caso de falha de rede, propaga AppError::NetworkError para que o chamador trate graciosamente (offline-first: nenhum dado parcial é salvo).
