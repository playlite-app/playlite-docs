Module fetch

Expand description

Busca e montagem dos dados técnicos de um jogo no PCGamingWiki.

Orquestra 7 queries sequenciais à Cargo API, cobrindo as tabelas: Infobox_game, API, Video, Input, Audio, L10n e Tags.

O rate limiting é aplicado automaticamente em cada query via cargo_query.
