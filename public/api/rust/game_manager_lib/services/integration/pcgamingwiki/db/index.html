game_manager_lib::services::integration::pcgamingwiki

Module db 

Source
Expand description

Persistência dos dados do PCGamingWiki no banco SQLite local.

Os dados são armazenados na tabela game_extras sem TTL — tratados como características fixas do jogo e atualizados apenas por invalidação explícita via invalidate_pcgw_data.
