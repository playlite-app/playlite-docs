Implementação do GameSource para a Ubisoft

Fontes de dados utilizadas:

  • %LOCALAPPDATA%\Ubisoft Game Launcher\settings.yaml Contém o caminho base onde os jogos estão instalados (misc.game_installation_path).

  • %LOCALAPPDATA%\Ubisoft Game Launcher\cache\configuration\configurations Arquivo binário com entradas YAML embutidas que lista todos os jogos da conta, incluindo nome, identificador e caminho relativo do executável.

Nota sobre jogos instalados: Para determinar se um jogo está instalado, verifica se sua pasta existe dentro de game_installation_path. O nome da pasta é inferido a partir do game_identifier (que normalmente corresponde ao nome da pasta de instalação).

Linux (Wine): No Linux, o diretório de dados é resolvido a partir do wine_prefix configurado. O caminho equivale a <prefix>/drive_c/users/<USER>/AppData/Local/Ubisoft Game Launcher. Se nenhum prefix for fornecido, a importação não estará disponível no Linux.
