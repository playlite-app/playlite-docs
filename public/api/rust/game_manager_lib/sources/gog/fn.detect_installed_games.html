Expand description

Marca como instalados os jogos cuja pasta existe dentro do diretório Games configurado do GOG Galaxy.

Heurística: o GOG Galaxy cria uma subpasta por jogo com o nome do produto exatamente como aparece na loja. Exemplo: “Tomb Raider I-III Remastered”

Observação: Alguns jogos na loja têm sufixos extras no nome (ex: “Leap of Faith + Official Walkthrough”), enquanto a pasta instalada usa apenas o nome base do jogo. Por isso o match verifica se o nome da loja começa com o nome da pasta, e não por igualdade exata.
