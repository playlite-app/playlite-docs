Comandos Tauri para Sistema de Recomendação v4.0

Faz JOIN com game_details para obter genres, tags categorizadas e series. Utiliza abordagem híbrida: perfil do usuário + collaborative filtering. Permite configuração de filtros (playtime), pesos personalizados, feedback (blacklist). Retorna razões detalhadas para cada recomendação.
