Comandos Tauri para obtenção e atualização de dados técnicos de jogos via PCGamingWiki.

Implementa um fluxo offline-first: dados são servidos do cache SQLite sempre que disponíveis, e buscados online apenas quando necessário. Falhas de rede são tratadas graciosamente — o frontend recebe None em vez de um erro.
