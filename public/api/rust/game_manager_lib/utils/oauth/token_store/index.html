Module token_store 

Source
Expand description

Armazenamento persistente de tokens OAuth2 por provedor.

Reaproveita a infraestrutura de secrets já existente (secrets.db + AES-256-GCM via security::encrypt/decrypt), guardando cada token serializado em JSON sob a chave oauth_<provider_id> na tabela encrypted_keys. Nenhuma tabela ou schema novo é necessário.
