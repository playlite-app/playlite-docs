1//! Módulos para integração com serviços externos.
2//!
3//! Fornece funcionalidades para interagir com APIs e serviços de terceiros.
4//! Cada módulo encapsula a lógica necessária para comunicação com um serviço específico,
5//! facilitando a manutenção e expansão do código.
6//!
7//! **Módulos:**
8//!
9//! - `gamebrain`: Integração com a API GameBrain.
10//! - `gamerpower`: Integração com a API GamerPower para busca de jogos grátis.
11//! - `gemini`: Integração com a API Gemini para funcionalidade de tradução com IA.
12//! - `itad`: Integração com a API IsThereAnyDeal para 'tracking' de preços e ofertas.
13//! - `pcgamingwiki`: Integração com a API PCGamingWiki para busca de informações sobre jogos.
14//! - `rawg`: Integração com a API RAWG para busca de jogos e tendências.
15//! - `steam`: Integração com a API Steam para obter detalhes e conquistas dos jogos.
16//! - `steamspy`: Integração com a API SteamSpy para estatísticas de jogos (median playtime).
17
18pub mod gamebrain;
19pub mod gamerpower;
20pub mod gemini;
21pub mod itad;
22pub mod pcgamingwiki;
23pub mod rawg;
24pub mod steam_api;
25pub mod steamspy;
