# Installation

Playlite already has releases available on GitHub.

## Download the latest version

Latest release:

- https://github.com/playlite-app/playlite/releases/latest

Or see all versions:

- https://github.com/playlite-app/playlite/releases

## How to install

1. Download the installer for your operating system from the Releases page.
2. Run the installer.
3. Open Playlite.

## Updating the app

When a new version is released:

1. Go to **Releases** on GitHub.
2. Download and install the new version.

> Since the app is local-first, updating usually keeps your data. Still, it is a good idea to make a **backup** before updating.

## Uninstalling

- Remove Playlite like any other application on your system.
