# API Reference - Playlite

Documentação técnica dos hooks e services do Playlite.

## Estrutura

- **Hooks**: Custom hooks React para gerenciamento de estado e lógica
- **Services**: Camada de comunicação com o backend Tauri

## Como usar esta documentação

Navegue pelos módulos no menu lateral para ver a documentação detalhada de cada hook e service.

Para informações gerais do projeto, consulte o [repositório no GitHub](https://github.com/Alan-oliveir/game_manager).
